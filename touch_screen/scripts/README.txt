
What to install in order to compile the touch_screen application for Android:
(for further references visit:
 http://doc.qt.io/qt-5/androidgs.html
 and:
 http://wiki.ros.org/android_ndk/Tutorials/Building%20The%20Example%20Applications%20using%20the%20Binary%20Distribution
 and:
 http://wiki.ros.org/android_ndk/Tutorials/SimpleNativeActivityApplication)

1. Qt5 for Android: download installer from Qt web page and select only
   to install "Android armv7". It will force you to install also qtcreator
   (even if you already have it).
   
2. Android SDK. Stand-alone version (no need to install Android Studio or
   extras)
   
3. Android NDK.

4. ROS for Android. 
   https://github.com/creativa77/roscpp_android/releases/download/v1.0/roscpp_android_ndk.tar.gz
