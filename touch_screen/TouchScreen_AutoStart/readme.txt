
Documentation
--------------------------------

There are two files in total:
	
	adb vision.docx
	virtualkeyboard.py
	
adb vision.docx introduces the use of the entire ADB, and the method of starting and starting and the installation and name of the software.

virtualkeyboard.py file is a computer-controlled tablet operating code.Through this you can use the computer to control the closing software of the tablet and control the background software to close and open.
